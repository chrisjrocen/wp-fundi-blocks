# wp-fundi-blocks
A WordPress plugin with Reusable gutenberg blocks to use on multiple sites.

## Blocks
- Fundi Carousel


## How to setup
- Run `npm install` to install all the sass + dev dependencies.
- Run `composer install` to get the PHP setup.
